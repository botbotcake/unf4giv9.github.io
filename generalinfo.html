General-Information

What is ApkEditor?

ApkEditor is an android app created for modifying android packages(Apps) on your mobile device to change them in order to suit your needs.


What does "full edit" do?

Full edit allows you to edit every part of an app individually. in ex.(Strings, Xml, Smali, drawables, and more.) some features require pro version like smali editing.


What does "Simple Edit" do?

Simple Edit allows you to replace resources inside of an app, such as an app icon an xml file, or an audio file.


Do I have to root my phone to use this?

For the most part, no but some features do require root to function such as edit data.


What does pro version have?

ApkEditor Pro version is highly reccomended, it comes with many useful features such as
• Smali Editing
• No ads
• Edit Data
• Android Manifest Editing
• Get Updates Faster
