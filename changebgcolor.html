How To: Change Background Color

Click Here For Another Guide

Step 1:


Open the apkeditor application.


Step 2:


Click on Select Apk From App, and select the app you would like to edit.


Step 3:


Select the option that says "full edit" and wait for everything to load up.


Step 4:


Click on the file icon and make your way to this file.
res -> Layout
Find the file you would like to change the background color in, for example something like activity_main.xml


Step 5:


inside of the first layout, if there is not already something that says android:background="@color/color"
Or
android:background="#hex"
then you need to add one of these, and specify your color.


Step 6:


You then need to click the save button, go back and select build in the top right and wait for it to complete. Once it completes, click remove -> Uninstall then Install it, and it should be good.
